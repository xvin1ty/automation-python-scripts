🤖 TRADING BOT SUITE - Quick Start
====================================

📁 FOLDERS:
-----------
📊 monte_carlo/     - Monte Carlo simulation results & reports
🤖 polymarket/      - Polymarket trading bot (prediction markets)
🔗 openclaw/         - OpenClaw integration (AI agent control)
📱 notifications/    - Telegram/Discord alert setup

🚀 QUICK START:
---------------

1. MONTE CARLO RESULTS (View First)
   → Open: monte_carlo/MONTE_CARLO_REPORT.md
   → Shows: Expected profits, risk analysis, recommendations

2. POLYMARKET BOT (Run This)
   → cd polymarket/
   → Edit .env with your settings
   → Run: python3 RUN_BOT.py

3. TELEGRAM ALERTS (Optional)
   → cd notifications/
   → Follow: SETUP_TELEGRAM.md
   → Edit: config.py with your token

4. OPENCLAW CONTROL (Advanced)
   → cd openclaw/
   → Run: python3 connect_trading_bot.py

⚠️  IMPORTANT:
--------------
• ALWAYS start with PAPER trading (fake money)
• NEVER risk more than you can afford to lose
• Set stop losses before trading
• Monitor daily

📞 NEED HELP?
-------------
